Week 1 課堂講義：數位工作區建立

今日目標是讓每位老師將基本環境建置完成，為後續五週實作打下穩健地基。請依序完成以下「今日要做的五件事」：

1. 建立 Google Drive 課程資料夾

於 Google 雲端硬碟建立主資料夾 北市大暑期地球科學課程_姓名，並在內建立 6 個子資料夾：

北市大暑期地球科學課程_姓名/ ├── week01_setup/ ├── week02_ground_shaking/ ├── week03_eq_location/ ├── week04_seismogram/ ├── week05_site_effect/ └── week06_final_project/

2. 建立 Google Form 測試表單

建立名稱為 Week 1 課程回饋與學習目標 的表單，包含：姓名、任職學校/領域、想學的地震主題、想學的數位工具、想帶回學校教的概念等 5 題。

3. 連接 Google Sheets 試算表

在表單的「回應」分頁點選「連結至試算表」，建立並確認表單回應能正常寫入該試算表中。

4. 測試 Google Apps Script (GAS)

在試算表中開啟「擴充功能」→「Apps Script」，貼上以下測試碼並執行與授權：

function courseHello() {
  SpreadsheetApp.getActiveSpreadsheet().toast("北市大暑期地球科學課程：GAS 測試成功！");
}

5. 建立 GitHub 個人 Repository

建立公開 Repo：earth-science-summer-2026-yourname，並初始化包含 README.md，寫下您的 100-200 字個人學習目標。

用途：蒐集上課同學的姓名、學校與 Gmail，並在填寫後自動寄送課程資源連結。

【做法 A：一鍵建置（建議）】

  1. 打開 Google Apps Script，建立新專案。
  2. 將 reference/gas/CourseRegistration.gs 內容全選貼上至編輯器中。
  3. 修改開頭 COURSE_CONFIG 中的網址與相關資訊，儲存專案。
  4. 下拉選單選擇 setupCourseRegistration 並點擊「執行」，完成 Google 權限驗證授權。
  5. 至 Google 雲端硬碟確認自動生成的表單與試算表。填寫測試表單，確認自動寄信正常。

【做法 B：手動逐步建立】

  1. 手動建立 Google 表單與題目，並連結至新的 Google 試算表。
  2. 開啟該試算表的「擴充功能」→「Apps Script」，貼上 CourseRegistration.gs 的代碼。
  3. 重要！請在專案中新增並執行以下安裝觸發器專用程式碼，避免重複建立表單資源：
function installTriggerOnThisSpreadsheet() {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  PropertiesService.getScriptProperties().setProperties({
    SPREADSHEET_ID: ss.getId(),
    COURSE_WEB_URL: 'https://github.com/oceanicdayi/Utaipei_2026_summer',
    COURSE_GITHUB_URL: 'https://github.com/oceanicdayi/Utaipei_2026_summer',
    EMAIL_SUBJECT: '【北市大暑期地球科學課程】歡迎加入｜課程資源連結',
    SENDER_NAME: '北市大暑期地球科學課程'
  });

  var triggers = ScriptApp.getProjectTriggers();
  for (var i = 0; i < triggers.length; i++) {
    if (triggers[i].getHandlerFunction() === 'onCourseFormSubmit') {
      ScriptApp.deleteTrigger(triggers[i]);
    }
  }

  ScriptApp.newTrigger('onCourseFormSubmit')
    .forSpreadsheet(ss)
    .onFormSubmit()
    .create();
  Logger.log('已在此試算表安裝表單提交後自動寄信觸發器。');
}

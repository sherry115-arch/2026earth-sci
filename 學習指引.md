# 2026 北市大暑期地球科學課程 - 學習指引

## 📚 課程概述

**課程名稱**：北市大暑期地球科學課程  
**課程期間**：2026年7月17日 - 8月21日（共6週）  
**目標對象**：中小學現職教師  
**課程主軸**：地震學與地球物理學  
**課程特色**：導入 AI/Agent 工具與雲端平台，產出可帶回學校使用的地震教學工具

---

## 🎯 學習目標

完成本課程後，學員將能夠：

1. **掌握地震基礎知識** - 了解地震成因、地震波、震度與規模等基本概念
2. **操作專業工具** - 熟悉地震監測設備、IRIS 教學活動、jAmaSeis 等軟體
3. **應用雲端技術** - 使用 Google 雲端工具（表單、試算表、Apps Script）進行教學自動化
4. **開發教學資源** - 設計並製作可在學校使用的地震教學工具
5. **整合 AI 工具** - 學習如何將 AI/Agent 工具應用於地球科學教學

## 🛠️ Week 1 環境建置詳細指南

### 1. 建立 Google Drive 課程資料夾

**步驟**：
1. 登入 Google 雲端硬碟
2. 建立主資料夾：`北市大暑期地球科學課程_姓名`
3. 在主資料夾內建立 6 個子資料夾：
   ```
   北市大暑期地球科學課程_姓名/
   ├── week01_setup/
   ├── week02_ground_shaking/
   ├── week03_eq_location/
   ├── week04_seismogram/
   ├── week05_site_effect/
   └── week06_final_project/
   ```

**用途**：
- 每週作業和資料的整理
- 方便教師批改和同學間分享
- 備份重要的課程資料

---

### 2. 建立 Google Form 測試表單

**步驟**：
1. 登入 Google 表單
2. 建立新表單，標題：`Week 1 課程回饋與學習目標`
3. 新增 5 個問題：
   - 姓名（短答案）
   - 任教學校/領域（段落）
   - 想學的地震主題（段落）
   - 想學的數位工具（段落）
   - 想帶回學校教的概念（段落）

**注意**：表單設定為「收集郵箱地址」以便後續聯繫

---

### 3. 連結 Google Sheets 試算表

**步驟**：
1. 在 Google 表單中，點擊「回應」分頁
2. 點擊 Google 表單圖示，建立新的試算表
3. 確認表單回應會自動填入試算表

---

### 4. 測試 Google Apps Script (GAS)

**步驟**：
1. 打開 Google 試算表
2. 點擊「擴充功能」→「Apps Script」
3. 貼上以下程式碼：

```javascript
function courseHello() {
  SpreadsheetApp.getActiveSpreadsheet().toast("北市大暑期地球科學課程：GAS 測試成功！");
}
```

4. 儲存並執行函數，應該會看到成功提示

---

### 5. 建立 GitHub 個人 Repository

**步驟**：
1. 登入 GitHub
2. 點擊「New」建立新 Repository
3. Repository 名稱：`earth-science-summer-2026-yourname`（將 yourname 替換為您的名字）
4. 設定為公開 (Public)
5. 新增 `README.md` 檔案
6. 在 README.md 中輸入 100-200 字的個人教學目標

**範例 README.md 內容**：
```markdown
# 地球科學暑期課程 - 個人教學目標

我參加這個課程的目標是：
- 學習地震學的基礎知識
- 掌握地震監測的數位工具
- 開發可在課堂上使用的地震教學活動
- 將 AI 工具應用於地球科學教學

期望能帶回學校的成果：
- 一套完整的地震教學教案
- 互動式的地震監測活動
- 學生可以參與的地震科學專案
```

## 💡 常見問題解答

### Q1: 我沒有程式設計背景，能參加這個課程嗎？
**A**: 當然可以！本課程從基礎開始，不需要程式背景。教師會逐步引導，並提供完整的範例程式碼。

### Q2: 如果我某週無法參加課程怎麼辦？
**A**: 所有課程資料和錄影會放在 GitHub 上，可以自行補課。建議盡量參加，以獲得最佳的學習效果。

### Q3: 我需要準備什麼硬體設備？
**A**: 基本上只需要一台可以上網的電腦（Windows 或 Mac）和智慧型手機。部分活動可能需要使用手機的感測器。

### Q4: 如何獲得技術支援？
**A**: 可以在課堂上直接詢問教師，或在 GitHub 的 Issues 中提出問題，教師和助教會盡快回覆。
